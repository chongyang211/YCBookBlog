## 06.韦根信号设计和原理

- Wiegand开发是什么：用于门禁系统和访问控制系统的数据传输协议。
- Wiegand开发准备事项有哪些：要掌握韦根传输协议的格式比如26位还是34位，韦根接线操作，数据处理和验证

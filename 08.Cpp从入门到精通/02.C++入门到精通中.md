## 03.数据和数据类型

### 3.1 变量和常量

#### 3.1.1 变量

**作用**：给一段指定的内存空间起名，方便操作这段内存。

**语法**：`数据类型 变量名 = 初始值;`

**示例**：

```cpp
#include <iostream>
using namespace std;

int main() {
    //变量的定义
    //语法：数据类型  变量名 = 初始值
    int a = 10;
    cout << "a = " << a << endl;
    return 0;
}
```

> 注意：C++在创建变量时，必须给变量一个初始值，否则会报错
